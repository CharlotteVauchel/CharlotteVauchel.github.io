Analyse de données

Projet de fin de session sur l’apprentissage de l’analyse de données avec python via des notebook Anaconda. Pour ce projet nous avons décidé de nous focaliser sur l’analyse de données d’un jeu de données pour s'entraîner que nous avons trouvé sur Kaggle. Dans un premier temps nous voulions utiliser l’API spotify mais celle-ci ne nous permettait pas de faire autant de requêtes que nous voulions pour tester le fonctionnement. Nous avons donc choisi un jeu de données sur Kaggle permettant d’analyser des données Spotify sur les genres des meilleurs musiques en fonction des années et leur nombre d'écoutes. Grâce à des librairies comme Numpy, pandas ou plotly nous avons pu créer deux graphiques analysant les statistiques Spotify.

2023

API cities

Création d’une API web en ASP .NET Core en C#. Pour cette API j’ai dû créer une base de données sur mariadb je l’ai ensuite mise dans un conteneur. Cette API permet d’accéder aux informations de la base de données.

La base de données contient des informations de plusieurs villes françaises, avec leur population et leurs coordonnées GPS. L’API contient plusieurs requêtes possibles, une première qui permet de renvoyer toutes les données de la base au format JSON et la seconde qui permet de renvoyer une liste de ville dans un périmètre donné autour d’une certaine longitude et latitude données.
